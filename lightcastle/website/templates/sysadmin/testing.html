Testing for Success

When setting up servers for deployment, Tamara sets up Jenkins to run tests before pushing code out. In addition, she arranges for the application to notify developers whether their code passes muster (often to the chagrin of the developers). For one open-source charity social-networking project, Tamara set up Jenkins to allow developers to know if their code is kosher.

{%include "stories/sa_stories.html"%}

Jenkins is an open source application for managing 'Continuous Integration' - a critical component of agile sofware development. 'Continuous Integration' means that you have a single location where all software changes are applied. Automated tests can then be executed on a regular schedule. This helps identify defects early in the process when they can be fixed with a minimum of effort.
