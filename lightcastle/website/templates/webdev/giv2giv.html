Rails development for social giving website Giv2giv

Giv2giv is a nonprofit that aims to make charitable giving a more social activity. When it needed to get a working prototype up and running, LightCastle was their go-to shop. The LightCastle team helped create several first-iteration, proof-of-concept RESTful apis establishing basic functionality and testing out technologies best suited to the task. LightCastle also set up continuous integration tests that ran each time new code was pushed to the codebase, ensuring a developer knew if his or her code was working as intended.

One example of a tool they brought to bear is Jenkins, a Continuous Integration framework. Through some creative scripting, Jenkins performed automated testing on each github commit, and report results in another tool, the giv2giv development team's BaseCamp Campfire chat room. Future efforts include further scripting to push github commits to the production environment upon successful Jenkins test completion. It is through use of a number of successful tools like Jenkins that LightCastle has helped move the giv2giv platform development forward with confidence and coordination. From direct experience of real-world problems creatively solved, I highly recommend LightCastle Technical Consultants.
